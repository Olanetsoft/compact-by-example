# Images Directory

## og-image.png

The Open Graph image used for social media sharing (Twitter, LinkedIn, Facebook, etc.).

**Current file**: `og-image.png`

**Recommended specifications**:
- **Dimensions**: 1200x630px (optimal for all platforms)
- **Format**: PNG or JPG
- **File size**: < 1MB (for fast loading)
- **Content**: Should include:
  - Site logo or branding
  - Title "Compact by Example"
  - Clear, readable text
  - High contrast for visibility

**Used in**:
- Global meta tags in `docs.json`
- Social sharing (Twitter Cards, Open Graph)
- Search engine previews

**Note**: Mintlify automatically overlays your logo, page title, and description on this background image when generating social media previews.

**Test your social cards**: https://metatags.io/
